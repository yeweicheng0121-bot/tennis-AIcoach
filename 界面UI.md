## 一、设计概述



### 1\.1 产品定位



AI 网球教练是一款面向业余网球爱好者的 AI 驱动训练助手。用户上传打球视频并关联 OPPO 手表数据，即可获得 NTRP 等级评定、技术/体能分析报告和个性化训练计划。



### 1\.2 设计目标



|目标|说明|
|---|---|
|**可信赖**|数据清晰、等级评定严谨，让用户感受到专业性|
|**易上手**|流程简单：上传 → 等待 → 查看报告，三步完成|
|**有动力**|清晰的进度反馈、量化的进步指标，激励用户坚持训练|
|**干净清爽**|大量留白、柔和色彩、无冗余信息|



### 1\.3 竞品参考



- **Swing Vision**：AI 网球视频分析标杆，但界面偏复杂

- **Strava**：运动数据可视化优秀，卡片设计值得借鉴

- **Whoop**：极简数据展示，黑绿配色高级

    

### 1\.4 差异化设计方向



|维度|竞品痛点|本产品策略|
|---|---|---|
|信息密度|功能堆砌，信息过载|单页只聚焦一个核心任务|
|数据可视化|复杂的雷达图、折线图|水平进度条 \+ 大数字，降低认知负担|
|流程引导|用户不知道下一步做什么|清晰的 CTA 按钮，单一路径引导|
|反馈机制|分析过程不可见|实时进度 \+ 步骤拆解，让等待有预期|





## 二、设计价值观



### 2\.1 核心原则



```Plain Text
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│   专注          →    每个页面只做一件事                         │
│   信任          →    数据清晰、进度透明                         │
│   激励          →    量化进步、正向反馈                         │
│   克制          →    只用必要的色彩和元素                       │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```



### 2\.2 设计关键词



|关键词|视觉表达|
|---|---|
|**专业**|严谨的排版、克制的色彩、精准的数据展示|
|**干净**|大量留白、纯白卡片、浅灰背景、细边框分割|
|**运动感**|翠绿色作为活力点缀、大号数字、进度动效|
|**温度**|柔和的圆角、温暖的绿色系、亲切的问候语|
